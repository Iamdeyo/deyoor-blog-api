Deyo Express Blog API

Endpoint Method Description Authentication
/api/auth/register POST Creates a new user account. No
/api/auth/login POST Login a user account. No
/api/auth/me GET Retrieves the currently logged-in user's information. Yes
/api/posts GET Retrieves a list of all blog posts. No
/api/posts/:id GET Retrieves a specific blog post by its unique ID. No
/api/posts/:id DELETE Delete a blog post using the ID. Yes
/api/posts POST Create a new blog post. Yes
/api/posts/:id PATCH Edit a blog post. Yes
